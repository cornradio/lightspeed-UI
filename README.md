# lightspeed-UI
windows quick launcher written in c# + using ahk for key binding

## 软件介绍
这是一个快速启动器,支持快速拖放新文件/快捷方式到快速文件夹，支持快捷键启动任意文件/快捷方式/文件夹

## 使用帮助
快捷键需要在鼠标点击桌面/任务栏之后按下触发  

lightspeed文件夹快捷键：↪ Alt + 0~9
打开程序的快捷： ↪ 0~9 + 快捷方式/文件夹首字母  
窗口前进后退：↪ Alt + 左右方向键  
刷新快捷键： F5  
右键 - v - M 可以设置成大图标模式
alt + / ： 快捷键列表查看
